# Flag images for language selector

Place PNG flag images here:

- `es.png` — Spanish flag (Spain or generic Spanish-language flag)
- `en.png` — English / UK or US flag

**Recommended size**: 32 × 24 px (or any 4:3 ratio), transparent background or solid.

The images are displayed at `height: 24px; width: auto` in the header.
They can be sourced from any free flag icon set (e.g. flagicons.lipis.dev).
